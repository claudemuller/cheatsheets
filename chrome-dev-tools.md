# Chrome Dev Tools

## Source
| Command         | Description                    |
|-----------------|--------------------------------|
| Ctrl + Shft + i | Bring up dev tools             |
| Ctrl + o        | Search for file                |
| Ctrl + Shft + o | Search for method              |
| Ctrl + F8       | Deactivate breakpoints         |
| Ctrl + b        | Set/remove breakpoint at caret |
